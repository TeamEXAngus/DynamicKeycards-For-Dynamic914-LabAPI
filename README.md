# DynamicKeycards-For-Dynamic914-LabAPI

Addon for [Dynamic 914](https://github.com/TeamEXAngus/Dynamic914-LabAPI) allowing for custom keycards to be used as inputs and outputs.

Requires [Dynamic Keycards](https://github.com/TeamEXAngus/DynamicKeycards-LabAPI).

View the example config [here](https://github.com/TeamEXAngus/DynamicKeycards-For-Dynamic914-LabAPI/blob/master/example-config.json).
